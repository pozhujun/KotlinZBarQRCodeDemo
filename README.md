** Kotlin 二维码扫描（Zbar）Demo**

ZBar 是款桌面电脑用条形码/二维码扫描工具，支持摄像头及图片扫描，支持多平台包括 iPhone 手机。同时 ZBar 提供了二维码扫描的 API 开发包。目前ZBar已经有大神完美移植到了Android平台，并且对安卓系统的不同CPU架构，以及碎片化的系统版本，都有了良好的支持和兼容。本文便是根据Github其中一个版本，进行Kotlin化的改造。
